TryTons Backend

The Fantasy TryTons League REST backend is running and available for runtime testing.

Available endpoints

  • Database health check GET /api/test/database
  • Players API GET /api/player
  • Clubs API GET /api/club
  • Positions API GET /api/position
Authentication login and registration use POST requests and should be tested through the frontend or Postman.

The database health check is restricted to authenticated administrators.
